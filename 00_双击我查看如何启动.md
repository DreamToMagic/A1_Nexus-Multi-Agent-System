# 🚀 A1_Nexus 快速启动指南 (Quick Start)

如果您忘记了如何启动系统，请查看以下说明：

## 💻 Windows 系统
**方式一：命令行模式 (推荐，最稳定)**
1. 找到当前文件夹下的 `start_windows.bat` 文件。
2. **双击运行**它。
3. 按照弹出的黑色窗口提示，输入 `1` (手动模式) 或 `2` (全自动挂机模式) 并回车即可。

**方式二：Web UI 图形界面模式 (新功能)**
1. 找到当前文件夹下的 `start_web_ui.bat` 文件。
2. **双击运行**它。
3. 等待黑色窗口提示“Running on local URL: http://0.0.0.0:8080”后，打开浏览器访问 `http://localhost:8080` 即可使用图形化控制台。

## 🐧 Linux / macOS 系统 (包括 ARM 架构)
**方式一：命令行模式 (推荐，最稳定)**
1. 打开终端 (Terminal)。
2. 使用 `cd` 命令进入当前文件夹（例如：`cd ~/Desktop/A1_Nexus_Portable`）。
3. 第一次运行前，需要给脚本添加执行权限（以后不需要再执行这步）：
   ```bash
   chmod +x start_linux_mac.sh
   ```
4. 运行启动脚本：
   ```bash
   ./start_linux_mac.sh
   ```
5. 按照终端提示，输入 `1` 或 `2` 选择模式并回车。

**方式二：Web UI 图形界面模式 (新功能)**
1. 打开终端 (Terminal)。
2. 第一次运行前，需要给脚本添加执行权限：
   ```bash
   chmod +x start_web_ui.sh
   ```
3. 运行启动脚本：
   ```bash
   ./start_web_ui.sh
   ```
4. 等待终端提示后，在浏览器中访问 `http://localhost:8080` (如果是远程服务器，请将 localhost 替换为服务器 IP)。

---

### 💡 进阶：命令行手动启动 (如果您不想用一键脚本)
如果您熟悉命令行，也可以直接使用 Python 启动：

*   **安装依赖 (仅首次需要)**:
    ```bash
    python auto_setup.py
    ```
*   **交互审批模式** (每完成一个任务会暂停等您确认):
    ```bash
    python nexus_core.py
    ```
*   **全自动挂机模式** (无人值守，自动跑完所有任务):
    ```bash
    python nexus_core.py --auto
    ```

---
*注：如果遇到任何问题，请确保您的电脑已安装 Python 3.8 或更高版本。*
